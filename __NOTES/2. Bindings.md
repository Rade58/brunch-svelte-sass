# Bindings

**THIS IS A "THING OF THE FORM ELEMENTS"**

## For text inputs

THIS IS WHAT WE ARE DOING WHEN WE DON'T USE BINDINGS; WE USE `on:input` EVENT HANDLER IKE THIS

```svelte
<script lang="ts">
  let inputData = ""
</script>

<input type="text" on:input={(e) => {inputData = e.currentTarget.value}}  />
<div>{inputData}</div>
```
